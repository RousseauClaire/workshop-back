workshop5 distanciel CRM Open Source

Lancé par NK Informatique le but de ce projet de workshop est la conception d'un CRM open source, ne comprenant que les fonctionnalités primaires et le plus facile à utiliser possible. Cette décision vient du constat que les nombreux CRM open source déjà disponible sur le marché souffrent de leur complexité et de leurs trop nombreuses fonctionnalités inutiles dans leur usage le plus courant.
